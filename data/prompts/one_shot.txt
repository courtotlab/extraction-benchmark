Please extract and return the following fields from the report document:

A. Clinical Testing Info
- Analysis Type — The type of analysis performed. For example: Variant analysis, Karyotyping, Microarray, etc.
- Sequencing Scope — The scope of the sequencing assay. Allowed values: "Gene panel", "Targeted variant testing", "Whole-Exome-Sequencing (WES)", "Whole-Genome-Sequencing (WGS)", "Whole-Transcriptome-Sequencing (WTS)"
- Tested Genes — The list of gene symbols that were tested in the assay
- RefSeq mRNA transcripts — The transcript identifier for the tested genes, in RefSeq format, e.g. "NM_000123.3"
- Sample Type — The type of sample used in the assay. For example: "Amplified DNA", "ctDNA", "Total DNA", "Total RNA"

B. Report Metadata
- Report Dates — The dates on which the samples were collected, received, verified, etc. Use the YYYY-MM-DD date format
- Report Type — The type of report. Allowed values: "Pathology" or "Molecular Genetics"
- Testing Context — The legal context in which the testing was performed: Allowed values: "Clinical" or "Research"
- Ordering Clinic — The name of the clinic who requisitioned the test. Include the city where the clinic is located when possible. (e.g. Mount Sinai Hospital (Toronto))
- Testing Laboratory — The name of the laboratory who performed the test. Include city if possible. (e.g. Ontario Institute for Cancer Research (Toronto))
- Reference genome — The reference genome assembly relative to which the variants are described (e.g. hg38)

C. Variant Details
- Variant ID — The OMIM, ClinVar, or dbSNP identifier for a variant. e.g. "VCV000055407.65"
- Gene Symbol — The HGNC symbol for the gene in which the variant is located. e.g. BRCA1
- Transcript ID — e.g., NM_000123.3
- HGVS descriptors — These descriptor strings encode the molecular basis of the variant in three contexts:
  - hgvsg: The genomic context (e.g. "g.43063903G>T"), 
  - hgvsc: The coding sequence context: (e.g. "c.5123C>A"), 
  - hgvsp: The context in the translated protein (e.g. "p.Ala1708Glu")
- Chromosome — the chromosome on which the variant is located, e.g. chr1, chr22, chrX, chrY
- Exon — The exon within the gene in which the variant is located
- Zygosity — Whether the variant was found in a homozygous, heterozygous, or compound heterozygous configuration.
- Interpretation — The ACMG clinical classification of the variant. Allowed values "Pathogenic", "Likely pathogentic", "Variant of uncertain significance (VUS)", "Likely benign", or "Benign"
- MAF — The minor allele frequency of the variant. 
- Type - The molecular type of the variant, e.g. frameshift, nonsense, synonymous, missense
